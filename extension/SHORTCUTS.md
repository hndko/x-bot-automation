# Phase 3: Keyboard Shortcuts

## Available Shortcuts

### During Execution

- **Space** - Pause/Resume
- **Escape** - Stop

### When Idle

- **Ctrl+D** - Delete Posts
- **Ctrl+R** - Undo Retweets
- **Ctrl+L** - Undo Likes

## How to Use

1. Open extension popup
2. Use shortcuts instead of clicking
3. Faster workflow!

**Note:** Shortcuts work when popup is focused.
